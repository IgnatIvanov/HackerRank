Louise и Richard играют в игру. Изначально у них есть кучка из палочек. Далее игроки ходят по очереди, Louise ходит первая. Во время своего хода игрок должен взять себе несколько палочек из кучки. Количество взятых палочек зависит от только текущего количества палочек в кучке, оно должно быть равно наибольшей степени двойки строго меньшей, чем текущее количество палочек в кучке.

Игра останавливается, когда в кучке остается ровно одна палочка. Тот, кто не смог сделать ход, проигрывает.

Вам задано число , выведите имя победителя в игре.

Формат входных данных

В первой строке записано целое число количество тестов. В каждой из следующих строк записано целое число .

Формат выходных данных

Для каждого теста выведите имя победителя в отдельной строке.

Ограничения


Пример входных данных

1
6

Пример выходных данных

Richard

Примечание

Первым ходом Louise возьмет палочки из кучки. Дальше Richard возьмет палочку, останется одна палочка. Louise не может сделать ход, поэтому она проиграла.
